# Customer Churn Analysis

## Business Problem
Customer churn directly impacts company revenue.  
The goal of this project is to identify key factors driving customer churn and provide actionable insights to reduce customer loss.

## Dataset
- Customer demographic data
- Service usage details
- Contract and payment information

## Tools Used
- Excel (initial analysis & dashboard)
- Python (Pandas, Matplotlib, Seaborn)
- Exploratory Data Analysis (EDA)

## Approach
1. Data cleaning and preprocessing
2. Exploratory Data Analysis to understand churn patterns
3. Identified high-risk customer segments
4. Visualized churn drivers using charts and dashboards

## Key Insights
- Customers with month-to-month contracts have higher churn
- Higher churn observed among customers with higher monthly charges
- Short-tenure customers are more likely to leave

## Business Recommendations
- Introduce long-term contract incentives
- Target high-risk customers with retention offers
- Improve onboarding experience for new customers
